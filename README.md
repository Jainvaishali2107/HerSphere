# 🌸 HerSphere

HerSphere is a full-stack women's wellness platform designed to provide a safe and supportive digital space. It offers AI-powered assistance, wellness tracking, and community features to help women manage their health and well-being.

## ✨ Features

- 🔐 Secure user authentication
- 🤖 AI wellness assistant
- 📅 Period tracker
- 👩 Anonymous community discussions
- 🚨 Emergency/SOS support
- 📱 Responsive and modern UI

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Backend:** Node.js, Express
- **Database:**  MongoDb
- **AI:** Google Gemini API

## 🚀 Getting Started

### Prerequisites

- Node.js

### Installation

```bash
npm install
```

Create a `.env.local` file and add your Gemini API key:

```env
GEMINI_API_KEY=your_api_key
```

Start the application:

```bash
npm run dev
```
